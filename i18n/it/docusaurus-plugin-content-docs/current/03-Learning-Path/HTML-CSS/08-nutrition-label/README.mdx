<TabItem value="commented" label="spiegato">

```html
<!DOCTYPE html>
<!-- Dichiara HTML5: deve sempre essere la prima riga -->

<html lang="en">
<!-- Tag radice; lang="en" segnala la lingua (SEO, accessibilità) -->

<head>
  <meta charset="UTF-8">
  <!-- Codifica UTF-8: supporto universale a lettere e simboli -->

  <title>Nutrition Label</title>
  <!-- Titolo che compare nella tab del browser -->

  <link href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800" rel="stylesheet">
  <!-- Importa il font 'Open Sans' in vari pesi da Google Fonts -->

  <style>
    /* ===== RESET BASE ===== */
    * {
      box-sizing: border-box;
      /* Padding e bordo inclusi nel calcolo di width/height */
    }

    html {
      font-size: 16px;
      /* 1rem = 16px, base per tutta la tipografia */
    }

    body {
      font-family: 'Open Sans', sans-serif;
      /* Font moderno, leggibile, senza grazie */
    }

    /* ===== CONTENITORE ETICHETTA ===== */
    .label {
      border: 2px solid black;
      /* Bordo nero come sulle vere etichette */
      width: 270px;
      /* Larghezza fissa simile alle etichette reali */
      margin: 20px auto;
      /* Centra la label orizzontalmente e dà spazio sopra/sotto */
      padding: 0 7px;
      /* Padding orizzontale per non far toccare il testo ai bordi */
    }

    /* ===== HEADER ===== */
    header h1 {
      text-align: center;
      /* Titolo centrato */
      margin: -4px 0;
      /* Togli spazio sopra/sotto, “schiaccia” il titolo in alto */
      letter-spacing: 0.15px;
      /* Leggerissimo tracking */
    }

    /* ===== PARAGRAFI E STRUTTURA RIGHE ===== */
    p {
      margin: 0;
      /* Niente spazio sopra/sotto */
      display: flex;
      justify-content: space-between;
      /* Distribuisce testo a sinistra e valori a destra su una riga */
    }

    /* ===== LINEE DI SEPARAZIONE ===== */
    .divider {
      border-bottom: 1px solid #888989;
      /* Riga sottile di colore grigio */
      margin: 2px 0;
      /* Spazio sopra e sotto la riga */
    }

    .bold {
      font-weight: 800;
      /* Testo molto spesso (simula titoli/nutrienti principali) */
    }

    .large {
      height: 10px;
    }

    .large, .medium {
      background-color: black;
      border: 0;
      /* Linea separatrice più spessa, nera, senza bordo esterno */
    }

    .medium {
      height: 5px;
    }

    .small-text {
      font-size: 0.85rem;
      /* Riduce la dimensione del testo per le note */
    }

    /* ===== SEZIONE CALORIE ===== */
    .calories-info {
      display: flex;
      justify-content: space-between;
      align-items: flex-end;
      /* Allinea quantità calorie in basso rispetto al testo */
    }

    .calories-info h2 {
      margin: 0;
    }

    .left-container p {
      margin: -5px -2px;
      /* Riduce lo spazio sopra/sotto per avvicinare “Calories” al numero */
      font-size: 2em;
      font-weight: 700;
    }

    .calories-info span {
      margin: -7px -2px;
      font-size: 2.4em;
      font-weight: 700;
    }

    /* ===== ALLINEAMENTO DEI VALORI ===== */
    .right {
      justify-content: flex-end;
      /* Allinea il contenuto tutto a destra */
    }

    .indent {
      margin-left: 1em;
      /* Rientro a sinistra per sotto-categorie (es: Saturated Fat) */
    }

    .double-indent {
      margin-left: 2em;
      /* Rientro doppio (es: Includes Added Sugars) */
    }

    /* ===== RIGHE GIORNALIERE ===== */
    .daily-value p:not(.no-divider) {
      border-bottom: 1px solid #888989;
      /* Linea separatrice solo alle righe che NON hanno la classe .no-divider */
    }

    .note {
      font-size: 0.6rem;
      margin: 5px 0;
      /* Nota finale più piccola e distanziata */
    }
  </style>
</head>

<body>
  <!-- ===== CONTENITORE ETICHETTA ===== -->
  <div class="label">
    <header>
      <h1 class="bold">Nutrition Facts</h1>
      <!-- Titolo principale grande, grassetto, centrato -->
      <div class="divider"></div>
      <!-- Riga separatrice grigia -->
      <p>8 servings per container</p>
      <!-- Numero di porzioni per confezione -->
      <p class="bold">Serving size <span>2/3 cup (55g)</span></p>
      <!-- Porzione consigliata (grassetto), il valore a destra -->
    </header>
    <div class="divider large"></div>
    <!-- Riga nera spessa, grande separazione visiva -->

    <div class="calories-info">
      <!-- Sezione “Amount per serving / Calories” -->
      <div class="left-container">
        <h2 class="bold small-text">Amount per serving</h2>
        <!-- Sottotitolo piccolo e in grassetto -->
        <p>Calories</p>

      <!-- Numero calorie grande e in evidenza -->
    </div>
    <div class="divider medium"></div>
    <!-- Riga nera meno spessa -->

    <div class="daily-value small-text">
      <!-- Inizio elenco % valori giornalieri -->
      <p class="bold right no-divider">% Daily Value *</p>
      <div class="divider"></div>
      <p><span><span class="bold">Total Fat</span> 8g</span> <span class="bold">10%</span></p>
      <p class="indent no-divider">Saturated Fat 1g <span class="bold">5%</span></p>
      <div class="divider"></div>
      <p class="indent no-divider"><span><i>Trans</i> Fat 0g</span></p>
      <div class="divider"></div>
      <p><span><span class="bold">Cholesterol</span> 0mg</span> <span class="bold">0%</span></p>
      <p><span><span class="bold">Sodium</span> 160mg</span> <span class="bold">7%</span></p>
      <p><span><span class="bold">Total Carbohydrate</span> 37g</span> <span class="bold">13%</span></p>
      <p class="indent no-divider">Dietary Fiber 4g</p>
      <div class="divider"></div>

      <!-- Sezione micronutrienti -->
```

**Cosa Ho Imparato:**

* Strutturare una nutrition label reale con HTML semantico e CSS
* Tipografia: font weight, dimensioni, allineamento e spaziatura
* Personalizzazione di linee, bordi e layout per un aspetto realistico
* Trucchi CSS per indentazione, stili dei divisori e allineamento dei numeri
* Come analizzare vere etichette nutrizionali e ricrearle da zero

**Tempo:**
2 ore e 35 minuti
